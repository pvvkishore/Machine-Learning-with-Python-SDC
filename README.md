## üöÄ Getting Started

### Prerequisites
- Python 3.8 or higher
- Jupyter Notebook or JupyterLab
- Basic understanding of Python programming

### Installation

1. Clone this repository:
   ```bash
   git clone https://github.com/pvvkishore/Machine-Learning-with-Python-SDC.git
   cd Machine-Learning-with-Python-SDC
   ```

2. Install required packages (once requirements.txt is available):
   ```bash
   pip install -r requirements.txt
   ```

3. Launch Jupyter Notebook:
   ```bash
   jupyter notebook
   ```

## üîç Notebook Descriptions

This repository uses an automated system to generate descriptions for all notebooks. To view descriptions of all course materials, see [NOTEBOOKS.md](NOTEBOOKS.md) (generated automatically when notebooks are added).

### For Instructors: Generating Descriptions

When you add new notebooks, run:
```bash
python notebook_analyzer.py . -o NOTEBOOKS.md
```

This will automatically analyze all notebooks and update the descriptions file.
