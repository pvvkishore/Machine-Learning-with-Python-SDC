## Research Papers

### Classic Papers Everyone Should Read
1. **"A Few Useful Things to Know About Machine Learning"** - Pedro Domingos
2. **"Random Forests"** - Leo Breiman
3. **"Support Vector Machines"** - Cortes and Vapnik
4. **"Deep Learning"** - Yann LeCun et al. (Nature, 2015)
5. **"Attention Is All You Need"** - Vaswani et al. (Transformers)

### Where to Find Papers
- **arXiv.org**: Pre-print repository
- **Google Scholar**: Search engine for papers
- **Papers With Code**: Papers with implementation
- **Semantic Scholar**: AI-powered paper search

### How to Read Papers
1. Read abstract and conclusions first
2. Understand the problem and motivation
3. Review figures and tables
4. Read methodology
5. Don't try to understand everything initially
