# Quick Start Guide

Get started with the Machine Learning course in 5 minutes!

## Step 1: Clone the Repository

```bash
git clone https://github.com/pvvkishore/Machine-Learning-with-Python-SDC.git
cd Machine-Learning-with-Python-SDC
```

## Step 2: Set Up Your Environment

### Option A: Using pip (Recommended)

```bash
# Create virtual environment
python -m venv ml_env

# Activate it
# On Windows:
ml_env\Scripts\activate
# On Mac/Linux:
source ml_env/bin/activate

# Install dependencies
pip install -r requirements.txt
```

### Option B: Using Conda

```bash
# Create environment
conda create -n ml_course python=3.9

# Activate it
conda activate ml_course

# Install dependencies
pip install -r requirements.txt
```

### Option C: Google Colab (No Installation)

1. Go to [Google Colab](https://colab.research.google.com)
2. Open notebooks from GitHub
3. Start learning immediately!

## Step 3: Verify Installation

```bash
# Start Jupyter Notebook
jupyter notebook

# Open notebooks/getting-started/welcome_notebook.ipynb
# Run all cells to verify everything works
```

## Step 4: Start Learning

### Week 1-2: Foundations

2. Review [SYLLABUS.md](SYLLABUS.md)
3. Explore `modules/01-introduction/`
