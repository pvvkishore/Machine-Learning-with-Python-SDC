# Module 5: Unsupervised Learning

## Overview
Discover hidden patterns in data without labels. Learn clustering and dimensionality reduction techniques.

## Learning Objectives
- Apply clustering algorithms to group similar data
- Evaluate clustering quality
- Reduce data dimensionality while preserving information
- Visualize high-dimensional data

## Topics
1. **K-Means Clustering**: Centroid-based clustering
2. **Hierarchical Clustering**: Dendrograms and linkage methods
3. **DBSCAN**: Density-based clustering
4. **Clustering Evaluation**: Silhouette score, elbow method
5. **PCA**: Principal Component Analysis
6. **t-SNE**: t-Distributed Stochastic Neighbor Embedding

## Key Files
- `kmeans_clustering.ipynb` - K-means implementation
- `hierarchical_clustering.ipynb` - Dendrogram visualization
- `dbscan_examples.ipynb` - Density-based clustering
- `pca_tutorial.ipynb` - Dimensionality reduction
- `tsne_visualization.ipynb` - High-dim data visualization
