# Module 7: Introduction to Deep Learning

## Overview
Enter the world of neural networks and deep learning. Learn to build and train neural networks for various tasks.

## Learning Objectives
- Understand neural network fundamentals
- Build neural networks with TensorFlow/Keras
- Apply CNNs for image classification
- Implement basic NLP tasks
- Use transfer learning

## Topics
1. **Neural Networks Basics**: Perceptron, activation functions, backpropagation
2. **Deep Learning Frameworks**: TensorFlow and Keras
3. **Building Models**: Sequential and Functional APIs
4. **CNNs**: Convolution, pooling, image classification
5. **NLP Basics**: Text preprocessing, embeddings, sentiment analysis
6. **Transfer Learning**: Using pre-trained models

## Key Files
- `neural_network_basics.ipynb` - Fundamentals
- `keras_tutorial.ipynb` - Building with Keras
- `cnn_image_classification.ipynb` - Image models
- `nlp_sentiment_analysis.ipynb` - Text classification
- `transfer_learning.ipynb` - Pre-trained models
